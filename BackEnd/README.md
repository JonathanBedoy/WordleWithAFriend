# Wordle With a Friend

### Back-End




#### Requirements:

- Requires the FrontEnd build to be put inside ``/public`` (more info on how to build the frontend located at ``../FrontEnd``).

#### Must Install dependencies:

```sh
npm install
```

### Runs the BackEnd

```
node ./app.js
```
